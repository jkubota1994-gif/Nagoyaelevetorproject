# Nagoya Subway Elevator Map

名古屋市営地下鉄の各駅におけるエレベーターの位置と、バリアフリールートを可視化するWebアプリケーションです。

## 概要

このプロジェクトは、名古屋市営地下鉄を利用する車椅子利用者、ベビーカー利用者、高齢者の方々が、スムーズに目的地まで移動できるように、エレベーターの設置場所をマップ上で直感的に確認できるツールを提供します。

### 主な機能

- 🚇 **エレベーター位置の可視化**: 駅周辺のエレベーター設置場所をピンで表示。
- 🕒 **リアルタイム次発案内**: 駅パネルから直近の列車出発時刻を確認可能。
- 🚶 **バリアフリールート検索**: 出発地からエレベーターを通るルートを算出し、徒歩・高齢者・タクシー料金の目安を表示。
- 🛰️ **OSRM & Nominatim 連携**: 最新の地図データに基づいた住所検索とルート案内。

## ディレクトリ構造

整理されたプロジェクト構造は以下の通りです：

- `index.html`, `main.js`, `style.css`: Webアプリの本体
- `data/`: Webアプリが読み込むデータセット (`timetable.js`, `all-elevators.js`)
- `scripts/`: データ抽出・加工用のPythonスクリプト群
- `raw_data/`: 元データ（Excel, 時刻表ZIPなど）
- `subway-timetable/`: 展開済みの時刻表データ

## 開発者向け情報

### データの更新方法

Pythonスクリプトを実行してデータを更新する場合は、`scripts/` フォルダ内のファイルを使用します。

```bash
# 例: 時刻表データの再生成
python scripts/generate_timetable_json.py
```

※ スクリプトは `raw_data/` フォルダ内のファイルを参照するように調整されています。
